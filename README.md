### Step 4: 
end the program
## Program:
```
#Program to find the rank of a matrix.
#Developed by: Mohamed rashith s
#RegisterNumber:24001082
import numpy as np
a = np.array([[3,2,5],[1,1,2],[3,3,6]])
c = np.linalg.matrix_rank(a)
print(c)

```
